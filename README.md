# LWC Draggable DataTable

This repository contains code for LWC component, which allows rows to reOrder. To demonstrate the drag-and-drop rows, it is using User object(which is common for all the orgs). 


To have a glimpse of the functionality. 
1. Download the code and push to scratch org. 
2. Open developer console. 
3. Open DefaultContainer app. (This is aura application). 
4. Click on "Preview" button, it'll open the application. 
5. Now, you can try to reOrder the rows and see the index is keeping updated each time. 
6. Click on submit button to get the JSON for further processing(with updated index).
